Hosts

The following are the list of available reports based on data collected for show hosts.

  • Appearance Summary

    This report provides a list of show hosts along with the number of appearances, their first and most recent appearance, for both regular and for all shows (including Best Of and Repeat shows).

  • Appearance Counts by Year

    This report provides a list of hosts and the number of appearances they have made each year.

  • Appearance Counts by Year: Grid

    This report breaks down the number of times a host has appeared on a show for a given year as a grid.

  • Debuts by Year

    This report provides a break down of when hosts made their debut on the show, grouped by year.
