Shows

The following are the list of available reports based on data collected for shows.

  • All Shows

    This reports provides a list of all shows that have been broadcasted, including Best Of and Repeat shows. The list does not include any pledge specials shows or special shows that were not aired in their original form.

  • All Women Panel

    This report provides a list of shows that have an all women panel.

  • High Scoring Shows

    This report provides a list of shows in which the total score for all three panelists is greater than or equal to 50, listed in descending order.

  • Highest Score Equals Sum of Other Scores

    This report provides a list of shows in which the highest panelist total score matches the sum of the two other panelists' scores.

  • Lightning Round Ending in a Three-Way Tie

    This report provides a list of shows in which all three panelists finished the Lightning Fill-in-the Blank segment in a three-way tie.

  • Lightning Round Starting and Ending in a Three-Way Tie

    This report provides a list of shows in which all three panelists started and ended the Lightning Fill-in-the Blank segment in a three-way tie.

  • Lightning Round Starting in a Three-Way Tie

    This report provides a list of shows in which all three panelists started the Lightning Fill-in-the Blank segment in a three-way tie.

  • Lightning Round Starting with Zero Points

    This report provides a list of shows in which a panelist started the Lightning Fill-in-the Blank segment with zero points.

  • Lightning Round with Zero Correct Answers

    This report provides a list of shows in which a panelist answered zero Lightning Fill-in-the Blank questions correct.

  • Low Scoring Shows

    This report provides a list of shows in which the total score for all three panelists is less than or equal to 30, listed in ascending order.

    One exception from the scoring is the 25th Anniversary Special that aired on October 27, 2018, which is excluded due to the unique Lightning Fill-in-the-Blank format.

  • Not My Job Guests vs Bluff the Listener Win Ratios

    This report compares the rates in which a Not My Job guest wins compared to how often a listener contestant chooses the correct Bluff the Listener story.

    For Not My Job guests, a win is considered when a guest answers 2 or more questions correct or when a scorekeeper grants an exception. The report also takes into account Not My Job guests that were only included in Best Of shows.

  • Original Shows

    This reports provides a list of original shows that have been broadcasted, which excludes Best Of and Repeat shows.

  • Panel Gender Mix

    This report provides a break down of the gender mix of panels for all broadcasted shows by year, excluding Best Of, Repeats, pledge specials, and the 25th Anniversary Special.

  • Search Shows by Multiple Panelists

    This report allows you to search for shows with a particular set of panelists (up to three) with options to include Best Of and Repeat shows.

    Shows that are Repeat Best Of shows will be included if either option is checked.

  • Show Counts by Year

    This report provides a count of the number of regular, Best Of and Repeat shows, grouped by year.

    The counts for the current year will only include shows that have already aired.

  • Show Descriptions

    This report provides a list of show descriptions for all shows.

  • Show Notes

    This report provides a list of show notes for all shows.

  • Shows with a Guest Host

    This report provides a list of all shows, including Best Of and Repeat shows, that have a guest host.

  • Shows with a Guest Scorekeeper

    This report provides a list of all shows, including Best Of and Repeat shows, that have a guest scorekeeper.

  • Shows with Perfect Panelist Scores

    This report provides a list of shows where a panelist finishes the Lightning Fill-in-the-Blank segment with a total score of 20 points, or as Bill Kurtis calls it, a "perfect score".

    The list also includes shows where a panelist has a total score greater than 20 points.
