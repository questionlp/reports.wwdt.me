  1. Panelists

Panelists

The following are the list of available reports based on data collected for panelists.

  • Aggregate Scores

    This report provides information on general statistics for the aggregate of panelist total scores and a break down of the number of times a total score has occurred.

  • Appearance Counts by Year

    This report provides a list of panelists and the number of appearances they have made each year.

  • Appearance Counts by Year: Grid

    This report provides a list of panelists and the number of appearances they have made each year as a grid.

  • Appearances by Year

    This report provides a list of appearances for a given panelist and a given year.

    Note: The previous form of this report is now available as "Appearance Counts by Year: Grid".

  • Average Scores by Year

    This report breaks down the average scores for a given panelist, broken out by year. The averages excludes scores from Best Of and Repeat shows; in addition to, scores from the show that aired on October 27, 2018 are also excluded, due to the unique scoring used.

  • Average Scores by Year: All

    This report breaks down the average scores for all panelists, broken out by year. The averages excludes scores from Best Of and Repeat shows; in addition to, scores from the show that aired on October 27, 2018 are also excluded, due to the unique scoring used.

  • Bluff the Listener Panelist Statistics by Year

    This report allows you to lookup the Bluff the Listener statistics for a panelist, including how many times their story was chosen and how many times they had the correct story, aggregated by year.

  • Bluff the Listener Statistics

    This report provides a break down of the number of times each panelist has had their Bluff the Listener story chosen, had the correct story, and the number of appearances on both regular shows that had a Bluff the Listener segment and Best Of shows with unique Bluff the Listener segments.

  • Bluff the Listener Statistics by Year

    This report provides a break down of the number of times each panelist has had their Bluff the Listener story chosen, had the correct story, and the number of appearances on both regular shows that had a Bluff the Listener segment and Best Of shows with unique Bluff the Listener segments for a given year.

    Counts include Bluff the Listener segments from regular shows and from Best Of shows with unique Bluff segments.

  • Debuts by Year

    This report provides a break down of when panelists made their debut on the show, grouped by year.

  • First Appearance Wins

    This report provides a list of panelists that have won outright or were tied for first place on their first appearance on the show.

  • First and Most Recent Appearances

    This report provides a list of panelists and their first appearance and most recent appearance on the show, for both regular shows and all shows, including Best Ofs and Repeats.

  • Highest Average Correct Answers by Year

    This report provides a list of the panelists with the highest average correct answers for a given year.

    Scoring information excludes Best Of and repeat shows.

  • Highest Average Scores by Year

    This report provides a list of the panelists with the highest average scores for a given year.

    Scoring information excludes Best Of and repeat shows.

  • Losing Streaks

    This report provides a break down of the longest losing streak for each panelist. "All Losses" includes shows in which a panelist did not finish in first place or tied for first.

  • Most Chosen Bluff the Listener Stories by Year

    This report allows you to lookup a list of panelists who have had the most chosen Bluff the Listener stories for a given year.

    Counts include Bluff the Listener segments from regular shows and from Best Of shows with unique Bluff segments.

  • Most Chosen Correct Bluff the Listener Stories by Year

    This report allows you to lookup a list of panelists who have had the most chosen Bluff the Listener stories, that were correct, for a given year.

    Counts include Bluff the Listener segments from regular shows and from Best Of shows with unique Bluff segments.

  • Most Correct Bluff the Listener Stories by Year

    This report allows you to lookup a list of panelists who have had the most correct Bluff the Listener stories for a given year.

    Counts include Bluff the Listener segments from regular shows and from Best Of shows with unique Bluff segments.

  • Most Wins by Year

    This report provides a list of the panelists with the most wins for a given year.

    Scoring information excludes Best Of and repeat shows.

  • Panelist vs Panelist

    This report allows you to view how well a panelist has faired against other panelists.

  • Panelist vs Panelist: All

    This report compares on how well each all panelists have faired against other panelists based on their scores and ranking, broken down my panelist.

  • Panelist vs Panelist Scoring

    A report that provides a way to lookup how well a panelist has scored against another panelist on a show-by-show basis.

    Best Of, Repeat and Special shows, including the 20th Anniversary Special that aired on October 27, 2018, will not be included in the results.

  • Perfect Score Counts

    This report provides a count for how many times a panelist that has had a total score of 20 points, which Bill Kurtis calls a "perfect score", or higher. The counts exclude any scores from Best Of and Repeat shows.

  • Rankings Summary

    This report provides statistics for the ranking of each panelist after each show.

  • Single Appearance

    This report provides a list of panelists that have made only one appearance on the show, excluding any Best Of and Repeat shows.

  • Statistics by Gender

    This report provides a break down of panelist statistics grouped by the gender that each panelist has identified themselves as.

  • Statistics Summary

    This report provides a break down of scoring statistics for each panelist.

  • Statistics Summary by Year

    This report provides a yearly break down of scoring statistics for a given panelist.

  • Win Streaks

    This report provides a break down of the longest winning streak for each panelist. "All Wins" includes shows in which a panelist did not finish in first place or tied for first.
