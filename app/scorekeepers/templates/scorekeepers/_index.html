Scorekeepers

The following are the list of available reports based on data collected for scorekeepers.

  • Appearance Summary

    This report provides a list of scorekeepers along with the number of appearances, their first and most recent appearance, for both regular and for all shows (including Best Of and Repeat shows).

  • Appearance Counts by Year

    This report provides a list of scorekeepers and the number of appearances they have made each year.

  • Appearance Counts by Year: Grid

    This report provides a list of scorekeepers and the number of appearances they have made each year as a grid.

  • Debuts by Year

    This report provides a break down of when scorekeepers made their debut on the show, grouped by year.

  • Scorekeeper Introductions

    This report provides a list of introductions that Bill Kurtis and other scorekeepers have used at the start of each show.
